# Suika-Board-SDK

このリポジトリは Suika-Board をキッティングするために必要な情報を収めたリポジトリです。

# 書き込むのに必要なもの

- Arduino (説明では Arduino UNO を用いて説明します)
- Suika Board
- Suika 書き込みケーブル

# ディレクトリ構成

```
.
│
│   # 各種手順書
├── doc
│   │   # Suika-Boardへの書き込み方法
│   ├── installation
│   │
│   │   # 音楽の打ち込み方法
│   └── score
│
│   # ソースコード
├── src
│
│   # 回路図
└── schema
```
